# Ecommerce Frontend

This project is a modern e-commerce frontend built with **React**, **Vite**, and **Tailwind CSS v4**.

## Prerequisites

- **Node.js**: `v20.19.0` or higher (recommended `v20.20.0`)
- **npm**: `v9` or higher

### Switch to the correct Node.js version

This project uses [nvm](https://github.com/nvm-sh/nvm). Before running any commands, please switch to the correct version:

```bash
nvm use
```
This will automatically use the version specified in the `.nvmrc` file.

## Getting Started

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode using Vite.  
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run build`

Builds the app for production to the `build` folder.  
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run serve`

Locally preview the production build.

## Technology Stack

- **Framework**: React 18
- **Build Tool**: Vite 7
- **Styling**: Tailwind CSS v4
- **State Management**: Redux Toolkit
- **Routing**: React Router v6
